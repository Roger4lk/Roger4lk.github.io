Friend(s) - Double Date Matching App

What This Project is:

Friend(s) is a social networking and double date matching app designed specifically for JMU students. The app facilitates connections between users looking for double date opportunities, creating a unique platform for social interaction and relationship building within the university community.

As a founding full-stack developer at Its Friend(s) Inc, I was instrumental in shipping the MVP of this innovative dating app in just 8 sprints. The app combines modern mobile development practices with robust backend services to deliver a seamless user experience.

Technical Implementation:

The app was built using React Native with NativeWind for styling, providing a native mobile experience across both iOS and Android platforms. The backend infrastructure leverages AWS services including Cognito for authentication and S3 for image storage, ensuring scalable and secure user management.

I implemented over 20 responsive screens and design-system components, which significantly improved the user interface quality and reduced UI defect rates by 30%. The comprehensive design system ensures consistency across the entire application while maintaining flexibility for future feature additions.

Backend Development:

On the backend, I created a robust Java-based architecture with controllers, DTOs, entities, mappers, repositories, and services. This layered approach provides clean separation of concerns and maintainable code structure. The integration of Cognito authentication and S3 image storage ensures secure user authentication and efficient media handling.

What I Learned:

This project provided invaluable experience in full-stack mobile development, from initial concept to production deployment. I gained deep expertise in React Native development, AWS cloud services integration, and the challenges of building a social networking platform. The experience of shipping an MVP in 8 sprints taught me the importance of rapid iteration, user feedback integration, and maintaining code quality under tight deadlines.
